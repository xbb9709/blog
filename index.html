不用VPN，轻松上Google

得益于我大天朝的“互联网长城”，才使得我们祖国的花朵免受“饕餮”的肆虐……
等等，我刚才是不是说了“饕餮”？

景甜好看！双眼皮好看！英语说得好听！yeah~yeah~yeah~yeah~yeah~

咳咳，言归正传。

临近期末，是不是有一堆课程论文等着你去拼凑，额，我是说，研究，对，研究。
于是你打开了百度学术（http://xueshu.baidu.com），看到没，知道你英语不好，主机名用的都是xueshu，而不是scholar！
打开之后是这样↓

这里还提供论文引用、论文互助，微信平台每天可以免费两次求助，用户体验还是挺棒的，如果不是特别冷门的论文的话，几分钟就能获得下载链接了。

其次就是谷歌学术（https://scholar.google.com）了，主页是这个样子↓

搜索页这个样子↓


所以，接下来的问题是，怎么上Google Scholar？

敲黑板！！！
第一步：下载这个玩意儿↓

hosts 点击下载

第二步：替换hosts

Windows 系统hosts位于 C:\Windows\System32\drivers\etc\hosts
Android（安卓）系统hosts位于 /system/etc/hosts
Mac（苹果电脑）系统hosts跟Linux一样位于 /etc/hosts
iPhone（iOS）系统hosts跟Linux Mac一样位于 /etc/hosts
Linux 系统hosts位于 /etc/hosts

按照对应的系统，把下载到的hosts复制到对应的文件夹，替换原有的hosts文件。

第三步：刷新DNS缓存
Windows
开始 -> 运行 -> 输入cmd -> 在CMD窗口输入

ipconfig /flushdns

Linux
终端输入

sudo rcnscd restart

对于systemd发行版，请使用命令

sudo systemctl restart NetworkManager

如果不懂请都尝试下

Mac OS X终端输入

sudo killall -HUP mDNSResponder

Android

开启飞行模式 -> 关闭飞行模式

通用方法

拔网线(断网) -> 插网线(重新连接网络)

接下来就享受Google、Gmail、维基百科、Twitter、Facebook、Youtube吧！

注：

Google、Gmail、维基百科、Twitter、Facebook等必须请用https加密方式打开。
（一般这些网站都是SSL加密链接）如谷歌学术：https://scholar.google.com/
如果不喜欢用谷歌香港，可以以 https://www.google.com/ncr 方式访问，(No Country Redirect)禁止国别跳转。

化而為鳥，其名為鵬。鵬之背，不知其幾千里也；

红叶与谁撷

清晨，烟雨迷蒙，驱车前往南部山区的红叶谷。
车程一小时，看着车窗上被风吹得斜下流淌的雨滴留下的水痕，耳机里单曲循环着这首《紅葉(もみじ)》，突然很想回家。

天气预报永远不准，说的是今天多云，现实是，天气预报部门一定是没有给老天爷交这个月的信息费，雨反而越下越大。

检票处“伞山伞海”，人虽然不少，但都打着伞，这就成了“斗伞”的竞技场了———谁特么管你排在前面还是排在后面，只要伞檐够尖、伞骨够强，戳死你没商量。
